# Calendar Button

This component represents a button used in the calendar, the intent of this 
component is that is is used in the header to change between the currently shown month. 

To style this component add a style object to your theme of the form

```js dark
{
  calendar: {
    button: {
      // Styles go here..
    }
  }
}
```
